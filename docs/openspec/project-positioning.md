# Project Positioning

## Product definition

`cursor-rules` 是一个以根目录 `.mdc` 文件为核心产物的规则库仓库。它的职责是：

1. 提供可直接放入 `.cursor/rules/` 的规则文件。
2. 提供一个静态、可筛选、可复制采用的规则目录站点。
3. 提供最低必要但足够清晰的控制文档，支持维护、收尾与交接。

## Non-goals

本仓库不承担以下职责：

- 不做模板仓库或脚手架。
- 不做“所有 AI 工具技巧”的知识库。
- 不引入与 `.mdc` 规则管理无关的复杂子系统。
- 不维护高噪声 changelog、零散实验笔记或重复性 FAQ。

## Archive-grade standard

“终局收敛”意味着：

- 文件少而硬，不靠额外层级制造复杂度。
- 公开入口、控制文档、站点目录之间不重复造数。
- 任何新增配置都必须解释其存在价值和维护成本。
