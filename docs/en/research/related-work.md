# Related Work

Cursor Rules sits between prompt collections, repository instruction files, and task-oriented skill systems. Its differentiator is that it treats reusable `.mdc` rules as a curated product surface rather than auxiliary project metadata.

## Comparison points

- Prompt collections optimize for copying snippets quickly.
- Repository instruction files optimize for repo-specific behavior.
- Skill systems optimize for task execution workflows.
- Cursor Rules focuses on reusable, reviewable engineering guidance.
